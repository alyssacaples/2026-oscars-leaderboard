## How to update the answers:

1. Open your Google Sheet: https://docs.google.com/spreadsheets/d/1laUmrg0B40zXrNriqToHZ91cQx44UvMn4bU78VjLSqo/edit?usp=sharing
2. At the bottom of the screen, click the **"+"** button to add a new tab.
3. Rename this new tab to exactly: **Winners**
4. In cell A1 type: **Category**
5. In cell B1 type: **Winner**
6. As the Oscars are announced, simply add rows to this "Winners" tab.
    * In column A, type the category name (e.g. `Best Picture` or `Actor in a Leading Role`). As long as it matches the beginning of what is in the form, the website will figure it out!
    * In column B, type the name of the winner (e.g. `Oppenheimer`).
7. The website leaderboard will automatically update within 60 seconds whenever you add a new winner to that sheet!
